# MainCrafts Website

A clean, responsive multi-page portfolio website for MainCrafts Technology. This project demonstrates modern HTML, CSS, and JavaScript for a small business landing site with a contact form, local submission storage, and theme toggling.

## Features

- Responsive multi-page layout: Home, About, Contact, and Submissions.
- Theme toggle for light/dark mode with persistence using `localStorage`.
- Contact form validation for name, email, and message input.
- Local submission storage and display on the Submissions page.
- Sticky navigation, mobile-friendly menu toggle, and back-to-top button.

## Files

- `index.html` - Homepage with service highlights and hero section.
- `about.html` - Company information and design/value proposition.
- `contact.html` - Contact form with validation and local storage support.
- `submissions.html` - Displays saved contact submissions from the browser.
- `styles.css` - Shared site styles, responsive layout, theme styles, and animations.
- `script.js` - Navigation toggle, theme persistence, form handling, and submission rendering logic.

## Getting Started

1. Open the `task 2` folder in your code editor.
2. Open `index.html` in a browser to view the site.
3. Use the navigation links to visit the About, Contact, and Submissions pages.

## Usage

- Click the moon/sun button to toggle dark mode.
- Submit the contact form on `contact.html`.
- Visit `submissions.html` to view saved messages stored locally in your browser.
